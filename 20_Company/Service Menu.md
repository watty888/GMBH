# Service Menu

## 1) Prompt QC Audit (1–2 weeks)
Deliverables:
- Failure mode inventory
- Proposed prompt spec improvements
- Minimal regression test set

## 2) Prompt Build + Validation (2–6 weeks)
Deliverables:
- Prompt spec (system + policies + tools)
- Test harness + scenarios
- Iteration log (what changed, why)
- Production readiness checklist

## 3) Ongoing Prompt QA Retainer (monthly)
Deliverables:
- Regression runs on changes (data updates, tool changes, model changes)
- Incident triage + mitigation
- Versioned prompt releases

## Optional add-ons
- Call transcript analysis & labeling
- Conversation flow redesign
- Tool contract design (“API for the agent”)
